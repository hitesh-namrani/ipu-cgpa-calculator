# **IPU CGPA Calculator**

A simple SGPA and CGPA calculator for GGSIPU students based on the official IPU grading scheme.
It converts marks into grade points and calculates accurate SGPA and CGPA using weighted averages.
The tool runs completely on the client side with no data storage or tracking.
Works on mobile and desktop with a clean dark-mode interface.
Developed by Hitesh Namrani.
